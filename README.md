# Tour of Heroes

A Hero listing application built using Angular. This project references the official Angular Tutorial, [Tour of Heroes](https://angular.io/tutorial/tour-of-heroes).

Topics covered:
* Component
* Service
* Routing
* Pipe
* CRUD Operations
* Angular CLI
* Angular In Memory Web API
